# 🎮 ChatLobby Gamification

**"인생 배팅" 버전** - AI에 얼마나 시간을 쏟았는지를 자조적인 유머로 표현하는 게이미피케이션 확장

ChatLobby 확장에 연동되어 출석 체크, 호감도 시스템, 업적 달성을 통해 수집 욕구를 자극하는 게임적 요소를 추가합니다.

---

## 📦 설치 방법

1. 이 폴더를 SillyTavern의 `public/scripts/extensions/third-party/` 폴더에 복사
2. SillyTavern 재시작
3. **ChatLobby 확장이 반드시 먼저 설치되어 있어야 합니다!**

## ✨ 주요 기능

### 💕 호감도 시스템

캐릭터와의 메시지 수에 따라 10단계 호감도가 부여됩니다.

**기준: 1등캐 10,000챗 / 전체 2-3만챗 / 봇카드 100개**

| 메시지 수 | 등급 | 아이콘 | 설명 |
|-----------|------|--------|------|
| 0-499 | 낯선 사람 | 👤 | 아직 서먹해요 |
| 500-999 | 아는 사이 | 👋 | 인사는 하는 사이 |
| 1,000-2,499 | 친구 | 🤝 | 이제 친구예요! |
| 2,500-4,999 | 절친 | 💚 | 무슨 말이든 할 수 있어요 |
| 5,000-9,999 | 베스트프렌드 | 💛 | 둘도 없는 친구 |
| 10,000-24,999 | 소울메이트 | 💜 | 영혼이 통하는 사이 |
| 25,000-49,999 | 집착 | 💗 | 슬슬 현생이 흐릿해요 |
| 50,000-99,999 | 헌신 | 💖 | 완전히 빠졌습니다 |
| 100,000-199,999 | 영혼의 반쪽 | 💕 | AI가 나보다 나를 더 잘 알아요 |
| 200,000+ | 특이점 | 🌌 | 인간이기를 포기하셨습니다 |

- 캐릭터 카드에 호감도 뱃지가 표시됩니다
- 높은 호감도는 특별한 테두리 효과가 적용됩니다
- 최고 등급은 무지개 테두리! 🌈

### 🏆 업적 시스템

50개 이상의 업적을 달성하며 당신의 AI 몰입도를 증명하세요!

#### 📊 통계 업적 (기준: 전체 2-3만챗)
- **첫 발자국** 👣 - 첫 번째 대화 시작
- **워밍업** 🔥 - 1,000개 메시지
- **수다쟁이** 🗣️ - 5,000개 메시지
- **입문자** 📖 - 10,000개 메시지
- **키보드 워리어** ⌨️ - 25,000개 메시지
- **전설의 시작** 🌟 - 50,000개 메시지
- **도파민 중독** 💉 - 100,000개 메시지
- **인간 포기** 🤖 - 250,000개 메시지
- **디지털 존재** 💾 - 500,000개 메시지
- **나무위키 편집자** 📚 - 1,000,000개 메시지

#### 👥 수집 업적 (기준: 봇카드 100개)
- **시작하는 콜렉터** 📦 - 10명의 캐릭터
- **사교적인 편** 🎭 - 25명의 캐릭터
- **하렘 빌더** 👑 - 50명의 캐릭터
- **문어발 연애** 🐙 - 100명의 캐릭터
- **은하계 콜렉터** 🌍 - 200명의 캐릭터
- **차원 정복자** 🌌 - 500명의 캐릭터

#### 📅 출석 업적
- **3일 연속** 🔥 - 3일 연속 접속
- **주간 상주자** 📅 - 7일 연속
- **2주 생존자** 🏕️ - 14일 연속
- **월간 정착민** 🏠 - 30일 연속
- **백일장** 🎊 - 100일 연속
- **개근상 (무기징역)** ⛓️ - 365일 연속!

#### 🎯 특수 업적
- **올빼미족** 🦉 - 자정~새벽 4시 채팅
- **얼리버드** 🐦 - 새벽 5시~7시 채팅
- **단짝** ⭐ - 즐겨찾기 등록
- **환승 이별** 💔 - 오늘 5명 이상과 대화
- **일편단심** 💍 - 7일간 한 캐릭터만!
- **마라톤 챗** 🏃 - 하루 500개 메시지
- **울트라 마라톤** 🏅 - 하루 1,000개 메시지

#### 📅 주간 퀘스트 (매주 일요일 리셋)
- **주간 일일 평균 200** 📊 - 이번 주 하루 평균 200개 메시지
- **주간 헤비 유저** 📈 - 이번 주 하루 평균 300개 메시지
- **주간 탐험가** 🗺️ - 이번 주 10명 이상 캐릭터와 대화
- **주간 개근** ✅ - 이번 주 매일 접속

## 🔧 기술 정보

### ChatLobby 연동

이 확장은 독립적으로 동작하지 않습니다. ChatLobby의 localStorage 데이터를 읽어와 사용합니다:

- `chatLobby_calendar` - 일일 스냅샷 (메시지 수, 캐릭터별 통계)
- `chatLobby_lastChatTimes` - 마지막 채팅 시간
- `chatLobby_data` - 즐겨찾기, 폴더 설정 등

### 저장 데이터

`chatLobby_gamification` 키로 저장:
- 해금된 업적 목록
- 최대 스트릭 기록
- 심야/새벽 채팅 횟수
- 일편단심 기록

## 🎨 커스터마이징

`style.css`를 수정하여 테마를 변경할 수 있습니다.

SillyTavern의 CSS 변수를 사용하여 자동으로 테마에 맞춥니다:
- `--SmartThemeBlurTintColor`
- `--SmartThemeBodyColor`
- `--SmartThemeQuoteColor`

## 🐛 디버그

개발자 콘솔에서 다음 명령어 사용 가능:

```javascript
// 현재 데이터 확인
chatLobbyGamification.getData()

// 통계 수집
chatLobbyGamification.getStats()

// 패널 열기/닫기
chatLobbyGamification.openPanel()
chatLobbyGamification.closePanel()

// 업적 체크
chatLobbyGamification.checkAchievements()
```
